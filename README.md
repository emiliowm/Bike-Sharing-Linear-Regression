# Predicting Bike Rental Demand: The Impact of Weather and Feature Engineering

This project investigates how environmental and temporal factors influence daily bike rentals. By implementing linear regression and applying advanced feature engineering, the model identifies key drivers behind cycling behavior.

## Project Overview
* **Objective:** Predict daily bike rental demand using the Bike Sharing Dataset.
* **Methodology:** Linear regression using the closed-form least squares solution.
* **Feature Engineering:** Implementation of polynomial (up to degree 3), logarithmic, and interaction terms to capture non-linear relationships.
* **Key Results:** * The baseline model achieved a Test MSE of 628,523.
    * The final engineered model reduced the Test MSE to 442,662 (a 30% improvement).
    * Temperature-related features were found to be the most informative predictors of rental volume.
